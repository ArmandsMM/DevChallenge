# DevChallenge
dev challenge about fetching simple list from api


Manual
1. $ pod init
2. $ open Podfile
3. edit podfile with needed pods

<code>
# Podfile
use_frameworks!

target 'DevChallenge' do
      pod 'RxSwift',    '~> 3.0'
      pod 'RxCocoa',    '~> 3.0'
end
</code>

4. $ pod install
5. Open DevChallenge.xcworkspace and build and run in a simulator or device.
